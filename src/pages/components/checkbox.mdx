## Examples

### Default checked

Use `defaultChecked` to set the initial state without controlling the component.

<ComponentSection>
  <ComponentExample demo="CheckboxCheckedExample">
    <CheckboxCheckedExample client:load />
  </ComponentExample>
</ComponentSection>

### Disabled

Use the `disabled` prop to prevent interaction.

<ComponentSection>
  <ComponentExample demo="CheckboxDisabledExample">
    <CheckboxDisabledExample client:load />
  </ComponentExample>
</ComponentSection>

### Controlled

Manage the checked state with React state using `checked` and `onCheckedChange`.

<ComponentSection>
  <ComponentExample demo="CheckboxControlledExample">
    <CheckboxControlledExample client:load />
  </ComponentExample>
</ComponentSection>

### Group

Use `CheckboxGroup` to manage multiple related checkboxes. Each child checkbox uses a `value` prop that matches the group's value array.

<ComponentSection>
  <ComponentExample demo="CheckboxGroupExample">
    <CheckboxGroupExample client:load />
  </ComponentExample>
</ComponentSection>

### Controlled group

Manage the group's selected values with React state using `value` and `onValueChange`.

<ComponentSection>
  <ComponentExample demo="CheckboxGroupControlledExample">
    <CheckboxGroupControlledExample client:load />
  </ComponentExample>
</ComponentSection>

### Parent checkbox

Use a parent checkbox with the `parent` prop to control a group of child checkboxes. Pass `allValues` to `CheckboxGroup` so the parent can toggle all children. The parent automatically shows an indeterminate state when some children are selected.

<ComponentSection>
  <ComponentExample demo="CheckboxGroupIndeterminateExample">
    <CheckboxGroupIndeterminateExample client:load />
  </ComponentExample>
</ComponentSection>
