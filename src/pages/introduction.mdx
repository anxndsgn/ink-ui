## Why Ink UI?

- **Ink UI** is a React component library built on top of [Base UI](https://base-ui.com) primitives and styled with [Tailwind CSS v4](https://tailwindcss.com). which provides a solid foundation of accessible and customizable components.
- The design focuses on warmth and approachability, with a color palette built around warm grays and oranges, and large rounded corners. The goal is to create interfaces that feel more human and inviting. **It is not suited for very professional or complex dashboard-style interfaces**, but rather for less complex, consumer-facing products that want to convey a sense of friendliness and delight.
- The motion system emphasizes smooth, natural transitions with a focus on micro-interactions that add delight without overwhelming the user. Subtle easing and thoughtful timing are used to create a sense of responsiveness and polish.

## Prerequisites

Before using Ink UI components, ensure your project is set up with:

- React 19
- Tailwind CSS 4
- `@base-ui/react`

## Installation

### 1. Add Tailwind and Base UI

Follow the [Tailwind CSS v4 installation guide](https://tailwindcss.com/docs/installation) and the [Base UI quick start](https://base-ui.com/react/overview/quick-start) for your framework.

### 3. Add utilities

Copy the `cn` helper into your project:

<CodeBlock code={`import { type ClassValue, clsx } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}`} lang="ts" />

### 4. Copy components

Browse the [Components](/components/button) section and copy any component source into your project. Each component is a self-contained file with no hidden dependencies.

## Dark mode

Dark mode is toggled by setting the `data-mode="dark"` attribute on the `<html>` element. All semantic tokens automatically adapt between light and dark contexts.
